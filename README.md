# Wander AI Studio

Personal educational AI dashboard created by Wander Martins.

## Purpose

Wander AI Studio is a command center for organizing teaching tools, AI platforms, prompts, educational projects, assessments, slides, games, and classroom resources.

## Main Modules

- IPC Studio
- Language Arts Studio
- Assessment Builder
- Game Builder
- Slides Studio
- Parent Message Assistant
- AI Platform Launcher
- Project Library

## Version

Current version: 1.0 prototype
